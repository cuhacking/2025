## Run Tests

```sh title="Terminal"
pnpm nx e2e docs-e2e --verbose
```

## Generate Tests (Codegen)

```sh title="Terminal"
pnpm exec playwright codegen
```

Select DESKTOP, TABLET, and/or MOBILE in the 'Projects' tab on the top left of the Playwright UI.
